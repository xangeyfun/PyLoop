Game Features

Core Mechanics

PyLoop is an idle clicker game where you accumulate "Lines of Code" (LOC) through clicking and automated upgrades. The game rewards patience and strategic upgrade purchases.

  • Clicking - Manually generate LOC by clicking the main button
  • Passive Income - Purchase upgrades that generate LOC automatically
  • Upgrades - Buy various upgrades to increase click value and passive generation
  • Multipliers - Boost your overall production with multiplicative upgrades

Upgrade System

The game features multiple tiers of upgrades that become available as you progress:

  • Keyboard Upgrade - Increases LOC per click
  • Script Automation - Provides passive LOC generation
  • AI Assistant - Dramatically boosts overall productivity
  • Cloud Infrastructure - Massive passive income multiplier

Progress & Persistence

Your progress is automatically saved to our servers. Create an account to:

  • Save your game progress across sessions
  • Access your account from any device
  • Compete on leaderboards (coming soon)
  • Never lose your progress

Technical Details

PyLoop is built using:

  • Frontend - Construct 3 for the game engine
  • Backend - Flask (Python) for server-side logic
  • Storage - JSON files for user data persistence
  • Security - Password hashing and session management
